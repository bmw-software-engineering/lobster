LOBSTER Python APIs
===================

Welcome to the LOBSTER API documentation.

Tools Python APIs
=================
These are Python interfaces for extracting requirements (e.g. from Codebeamer or TRLC ), extracting code traces (e.g. from Python), extracting tests (e.g. from cpptest, JSON) and generating traceability report as per user defined tracing policy.
This section documents the Python APIs for LOBSTER tools.
