Tools Python API
=================

This section documents the Python APIs for all LOBSTER tools found under ``lobster.tools``.
All tools share a common setup and CLI/runtime base provided in ``lobster.common``
(e.g., ``MetaDataToolBase``, ``LOBSTER_Per_File_Tool``, ``items``, ``location``, and ``io``).
Referencing and configuration patterns are consistent across tools.
